# 🛍️ Fancy Bags - E-commerce Website

An online bag store web application built using HTML, CSS, JavaScript, PHP, and MySQL.
It allows users to view, add, update, and delete bags with a clean and interactive interface.
## 🚀 Features
- Product listing (bags collection)
- Product images gallery
- Product detail pages
- PHP backend 

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
